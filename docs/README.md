## Building the Documentation

This documentation can be built into a static site using [MkDocs](https://www.mkdocs.org/):

```bash
# Install MkDocs
pip install mkdocs

# Build the documentation
mkdocs build

# Serve the documentation locally
mkdocs serve
```
